## Stats Tracker

Search any Deadlock player by Steam64 ID, Steam32 ID, or Steam vanity name. See:
- Rank tier (Obscurus → Eternus) with matching badge colour
- Win rate, W/L record, total matches, avg KDA
- Per-hero stats breakdown sorted by most played
- Last 15 match history with hero, KDA, result, duration and date
